# 人脸记忆校准器说明

FurColor 的“记忆”不是自行生成标签的黑盒训练。摄影师在复核页把候选标为 `human` 或 `fursuit` 后，系统保存特征和标签；当两类各至少有 2 个样本时，自动重训一个位于项目私有目录的逻辑回归校准器。

- YuNet 仍负责产生候选；校准器只修正候选属于真人脸的概率。
- 强几何证据继续作为安全门，学习模型不能单独决定公开交付。
- 训练准确率不是验证准确率。不同会场、灯光、兽装造型和遮挡必须持续人工复核。
- `face_labels.jsonl` 与 `face_memory.json` 包含敏感派生数据，已被发布流程排除，不得上传。
- 商业交付前必须完成人工隐私确认，不能把模型输出视为授权或合规结论。
