# Privacy Policy for FurColor Studio

## 默认原则

FurColor Studio 采用 local-first 设计。真实照片、文件路径、人脸候选、人工反馈、眼睛标注、水印、模型权重和项目数据库默认只保存在操作者本机。

## 明确不会上传的内容

- 原片、草稿和交付成片；
- 客户姓名、联系方式、活动名单和授权书；
- 人脸框、误检反馈、兽装眼睛标注；
- 本机绝对路径、水印源文件；
- API 密钥、SSH 私钥、服务器凭据；
- 训练集、记忆库与模型权重。

## 云端演示版

云端演示版不提供文件上传、路径读取、照片处理、交付生成或管理接口。容器不挂载宿主照片目录，并以非 root 用户和只读根文件系统运行。

## 操作者责任

操作者必须确认拍摄和处理权限，遵守活动方与圈内隐私规范，并在公开发布或商业交付前人工检查真人脸、反射、胸牌、二维码、地理位置等识别信息。
